# Azure Synapse Workspace Project  

This project contains a full Azure Synapse Analytics workspace export, including 
pipelines, datasets, linked services, and dataflows created within Synapse Studio.  

It demonstrates how Synapse can orchestrate ETL workflows end-to-end 
using its built-in integration features 

**Includes:**  
- Synapse pipelines (.json)  
- Datasets and linked services for data sources and sinks  
- Dataflows for transformations  
- SQL scripts for analytical processing  

**Tech Stack:**  
Azure Synapse Analytics | Azure SQL Database | Azure Storage | ETL Pipelines | Dataflows
